# BabaCafe
秋田県三種町のBabaCafeアプリ

## docker-compose
### ローカル開発環境(M1,M2のMac, ローカルのnode環境使用)
```shell
$ docker-compose -f docker-compose.mac_arm64.yml up -d --build
```

### ローカル開発環境(x86_64, コンテナのnode環境使用)
```shell
$ docker-compose -f docker-compose.yml -f docker-compose.develop.yml up -d --build
```

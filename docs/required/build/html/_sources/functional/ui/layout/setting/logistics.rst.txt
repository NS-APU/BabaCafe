物流業者向け物流設定画面
===============================================================

        :align: center

        物流業者向け物流設定画面

        :header-rows: 1
        :align: center

        * - 識別ID
          - 項目名
          - 種別
          - 説明
        * - ①
          - 集荷・配送方法の設定領域の見出し
          - ラベル
          - 集荷・配送方法の設定領域を示す見出し。
        * - ②
          - 集荷・配送方法の設定欄
          - ラベル
          - 設定されている集荷・配送方法。

            * 初期値は「巡回経路で集荷や配送を行う」

        * - ③
          - 集荷・配送方法変更ボタン
          - ボタン
          - 集荷・配送方法の変更をするボタン。

            * ボタンを押下した場合、集荷・配送方法変更ダイアログを開く

          - 巡回経路設定領域見出し
          - ラベル
          - 巡回経路の設定領域を示す見出し。

          - ボタン

          - 便アコーディオン
          - アコーディオン
          - 登録されている便の設定を開閉するアコーディオン。

            * 押下することでアコーディオン内コンテンツの展開または折りたたみを行う

        * - ⑩
          - 車両設定領域の見出し
          - ラベル
          - 車両に関する設定をする領域を示す見出し。
        * - ⑪
          - 衝撃設定欄
          - ラベル
          - 設定されている作物が配送に耐えられる衝撃度合の目安。
        * - ⑫
          - 最大取扱い量設定欄
          - ラベル
          - 設定されている最大取扱い量。
        * - ⑬
          - 時刻表設定領域の見出し
          - ラベル
          - 時刻表に関する設定をする領域を示す見出し。
        * - ⑭
          - 時刻表
          - テーブル
          - 設定されている時刻表。
        * - ⑮
          - 便追加ボタン
          - ボタン
          - 便の追加をするボタン。

            * ボタンを押下した場合、便追加ダイアログを開く

        * - ⑯
          - 便編集ボタン
          - ボタン
          - 便の編集をするボタン。

            * ボタンを押下した場合、便編集ダイアログを開く

        * - ⑰

          - ボタン
          - 便の削除をするボタン。

            * ボタンを押下した場合、便削除ダイアログを開く

集荷・配送方法変更ダイアログ
---------------------------------------------------------------

.. topic:: 画面レイアウト

        :align: center

        集荷・配送方法変更ダイアログ

.. topic:: 構成一覧

    .. list-table:: 集荷・配送方法変更ダイアログ
        :header-rows: 1
        :align: center

        * - 識別ID
          - 項目名
          - 種別
          - 説明
        * - ①
          - 変更メッセージ
          - ラベル
          - 集荷・配送方法の変更を促すメッセージ。
        * - ②
          - 集荷・配送方法選択欄
          - ラベル＋ラジオボタン
          - 集荷・配送方法を選択するラベルとラジオボタン。

            * 必須入力
            * 選択肢：巡回経路で集荷や配送を行う、集荷先や配送先に直接伺う

        * - ③
          - キャンセルボタン
          - ボタン
          - 集荷・配送方法の変更をキャンセルするボタン。

            * キャンセルボタンを押下した場合、ダイアログを閉じる

        * - ④
          - 変更ボタン
          - ボタン
          - 集荷・配送方法の変更を処理するボタン。

            * 追加ボタンを押下した場合、集荷・配送方法の変更を処理し、ダイアログを閉じ、物流業者向け物流設定画面の設定を最新化する

便追加ダイアログ
---------------------------------------------------------------

.. topic:: 画面レイアウト

    .. figure:: ../../img/便追加ダイアログ.png
        :align: center

        便追加ダイアログ

.. topic:: 構成一覧

    .. list-table:: 便追加ダイアログ
        :header-rows: 1
        :align: center

        * - 識別ID
          - 項目名
          - 種別
          - 説明
        * - ①
          - 追加メッセージ
          - ラベル
          - 便の追加を促すメッセージ。
        * - ②
          - 便名入力欄
          - ラベル＋テキストエリア
          - 便名入力欄のラベルとテキストエリア。

            * 必須入力
            * 入力可能文字数：最大50文字まで

        * - ③
          - 車両設定領域の見出し
          - ラベル
          - 車両に関する設定をする領域を示す見出し。
        * - ④
          - 衝撃選択欄
          - ラベル＋コンボボックス
          - 作物が配送に耐えられる衝撃度合の目安を選択する欄のラベルとコンボボックス。

            * 必須入力
            * 選択肢：とても強い、強い、弱い、とても弱い

                | ※衝撃値は以下のように定義する

                * とても強い：90G
                * 強い：60G
                * 弱い：45G
                * とても弱い：30G

        * - ⑤
          - 最大取扱い量入力欄
          - ラベル＋テキストボックス
          - 取扱い可能な最大の予約数量を入力する欄のラベルとテキストボックス。

            * 必須入力
            * 入力可能文字数：最大3桁まで
            * 入力可能な文字：半角数字
            * 入力範囲：1～100
            * 入力欄の末尾に「予約」を表示する

        * - ⑥
          - 時刻表設定領域の見出し
          - ラベル
          - 時刻表に関する設定をする領域を示す見出し。
        * - ⑦
          - CSVインポートボタン
          - ファイル選択ボタン
          - CSVファイルのファイル選択ボタン。

            * 選択できるファイルはCSV形式であること
            * 所定の様式で記載されたCSVファイルであった場合にはCSVデータを時刻表に設定を反映する
            * 所定の様式で記載されていないCSVファイルの場合はトーストでエラーを表示する

        * - ⑧
          - 時刻表
          - テーブル
          - 設定する時刻表のデータを表示する。
        * - ⑨
          - キャンセルボタン
          - ボタン
          - 便の追加をキャンセルするボタン。

            * キャンセルボタンを押下した場合、ダイアログを閉じる

        * - ⑩
          - 追加ボタン
          - ボタン
          - 便の追加を処理するボタン。

            * 必須入力項目がすべて入力されている場合のみ押下が可能
            * 追加ボタンを押下した場合、便の追加を処理し、ダイアログを閉じ、物流業者向け物流設定画面の設定を最新化する

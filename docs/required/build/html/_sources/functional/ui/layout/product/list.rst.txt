===============================================================
出品一覧画面
===============================================================

.. topic:: 画面レイアウト

    .. figure:: ../../img/出品一覧画面.png
        :align: center

        出品一覧画面

.. topic:: 構成一覧

    .. list-table:: 出品一覧画面
        :header-rows: 1
        :align: center

        * - 識別ID
          - 項目名
          - 種別
          - 説明
        * - ①
          - 出品ボタン
          - ボタン
          - 出品画面に遷移するボタン。

            * 押下すると出品画面に遷移する

        * - ②
          - 出品アイテム
          - カード＋リンク
          - 出品アイテムを表示するカードリンク。

            * 押下すると該当する出品の出品詳細画面に遷移する
            * 出品日時の降順（新しい順）に表示する
            * 出品が取り消されたアイテムはグレイ表示にする
            * 出品期間が過ぎたアイテムはグレイ表示にする

        * - ③
          - 生産者アイコン
          - アイコン
          - 生産者の区分を示すアイコン。

            * 生産者が個人か法人かでアイコンを変える

        * - ④
          - 生産者名ラベル
          - ラベル
          - 生産者の名前を表示するラベル。
        * - ⑤
          - 作物名ラベル
          - ラベル
          - 出品された作物の名称を表示するラベル。
        * - ⑥
          - 出品画像
          - イメージ
          - 出品されたときにアップロードされた画像を表示するイメージ領域。

            * 画像が無い場合にはデフォルト画像を表示する

        * - ⑦
          - 単位ラベル
          - ラベル
          - 出品された作物の単位を表示するラベル。
        * - ⑧
          - 金額ラベル
          - ラベル
          - 出品された作物の名称を表示するラベル。
        * - ⑨
          - 予約期間ラベル
          - ラベル
          - 出品された作物の予約期間（開始日、終了日）を表示するラベル。

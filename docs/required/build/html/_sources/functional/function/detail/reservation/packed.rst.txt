---------------------------------------------------------------
作物出荷
---------------------------------------------------------------

.. list-table:: 機能詳細：作物出荷
    :align: center

    * - 機能No.（業務要件No）
      - 3-5
    * - 機能分類（ビジネスプロセス1）
      - 予約管理
    * - 機能名称（ビジネスプロセス2）
      - 作物出荷
    * - 　　アクター、トリガー
      - アクターは予約詳細画面から出荷ボタンを押下する
    * - 　　概要
      - 生産者が予約された作物の出荷をしたことを連絡する
    * - 　　事前条件
      - 予約詳細画面を表示していること
    * - 　　基本フロー
      - #. システムは確認ダイアログを表示する
        #. アクターは配送者を選択する（配送者の選択肢は自身と物流業者として登録されているアカウントの一覧）
        #. アクターは配送手配ボタンを押下する
    * - 　　代替フロー
      - なし
    * - 　　例外フロー
      - * 確認ダイアログでキャンセルボタンが押下された場合、システムは確認ダイアログを閉じる
    * - 　　事後条件
      - * システムは予約内容のステータスを集荷配送中の状態にする
        * システムは予約詳細画面を表示する
        * システムは出荷が済んだことをメッセージで表示する
    * - 補足
      - * 自身が出品した生産者の場合のみ操作できる
        * 予約内容のステータスが出荷準備中の場合のみ操作できる

項目名

種別

説明

①

変更メッセージ

ラベル

集荷・配送方法の変更を促すメッセージ。

②

集荷・配送方法選択欄

ラベル＋ラジオボタン

集荷・配送方法を選択するラベルとラジオボタン。

  • 必須入力

  • 選択肢：巡回経路で集荷や配送を行う、集荷先や配送先に直接伺う

変更ボタン

ボタン

集荷・配送方法の変更を処理するボタン。

  • 追加ボタンを押下した場合、集荷・配送方法の変更を処理し、ダイアログを閉じ、物流業者向け物流設定画面の設定を最新化する

ラベル＋テキストエリア

便名入力欄のラベルとテキストエリア。

  • 必須入力

  • 入力可能文字数：最大50文字まで

③

車両設定領域の見出し

ラベル

車両に関する設定をする領域を示す見出し。

④

衝撃選択欄

ラベル＋コンボボックス

作物が配送に耐えられる衝撃度合の目安を選択する欄のラベルとコンボボックス。

  • 必須入力

  • 選択肢：とても強い、強い、弱い、とても弱い

    ※衝撃値は以下のように定義する

    • とても強い：90G

    • 強い：60G

    • 弱い：45G

    • とても弱い：30G

⑤

最大取扱い量入力欄

ラベル＋テキストボックス

取扱い可能な最大の予約数量を入力する欄のラベルとテキストボックス。

  • 必須入力

  • 入力可能文字数：最大3桁まで

  • 入力可能な文字：半角数字

  • 入力範囲：1～100

  • 入力欄の末尾に「予約」を表示する

⑥

時刻表設定領域の見出し

ラベル

時刻表に関する設定をする領域を示す見出し。

⑦

CSVインポートボタン

ファイル選択ボタン

CSVファイルのファイル選択ボタン。

  • 選択できるファイルはCSV形式であること

  • 所定の様式で記載されたCSVファイルであった場合にはCSVデータを時刻表に設定を反映する

  • 所定の様式で記載されていないCSVファイルの場合はトーストでエラーを表示する

⑧

時刻表

テーブル

設定する時刻表のデータを表示する。

⑨

キャンセルボタン

ボタン

  • キャンセルボタンを押下した場合、ダイアログを閉じる

⑩

  • 必須入力項目がすべて入力されている場合のみ押下が可能

  • 追加ボタンを押下した場合、便の追加を処理し、ダイアログを閉じ、物流業者向け物流設定画面の設定を最新化する
